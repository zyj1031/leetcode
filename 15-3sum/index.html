# 15. 3Sum

Given an array nums of n integers, are there elements aaa, bbb, ccc in nums such that a+b+c=0a + b + c = 0a+b+c=0? Find all unique triplets in the array which gives the sum of zero.

Notice that the solution set must not contain duplicate triplets.
